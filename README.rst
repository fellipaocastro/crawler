crawler
=======

Crawler

Requirements
------------

.. code-block:: bash

    Python 2.7.8
    pip 1.5.6

Setup
-----

.. code-block:: bash

    $ pip install -r requirements.txt

Run
---

.. code-block:: bash

    $ ./crawler.py

Test
----

.. code-block:: bash

    $ ./test.py

Source code check
-----------------

.. code-block:: bash

    $ flake8 .
